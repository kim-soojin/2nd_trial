# 2nd_trial
오늘의 긍정확언 표시

# 목적
오픈API 를 활용해 Vanilla JS로 매일매일 확인할 수 있는 **긍정확언** 서비스 제공

# 개발환경
- Front : HTML, CSS, JavaScript
- OPEN API : [fav.Qs](https://favqs.com/api)

# 출처
- OPEN API : [fav.Qs](https://favqs.com/api)

# 구현할 기능 목록
- 반응형으로 디자인 적용할 것
- 마우스 호버 이펙트 적용
- 로그인 기능은 없으나, 좋아요는 표시 할 수 있도록 할 것
